# 식빵(食パン)으로 만드는 요리 — 정밀 분석

일본 슈퍼 어디에나 있는 **식빵(食パン)** 하나로 만들 수 있는 요리들의 **핵심 원리·실패 포인트·재료 팁**을 정리했다. 앱(양식 카테고리)에도 레시피로 추가.

> 식빵 두께: 일본은 4枚切り(두꺼움)~8枚切り(얇음)로 나뉜다. **토스트·프렌치토스트는 4~6枚切り(두꺼움)**, **식빵피자·샌드는 6~8枚切り(얇음)**가 잘 맞는다.

## 1. 전남친 토스트 (クリームチーズ&ジャムトースト)
**유래:** 2018년 "전 남친이 해주던 토스트가 먹고 싶어 연락해 물어봤다"는 사연으로 화제가 된 초간단 레시피.
**구성:** 구운 식빵 + 크림치즈 + 블루베리 잼 + 전자레인지 10초.
**핵심:**
- **뜨거운 식빵 × 차가운 크림치즈**의 온도 대비가 맛의 정체성. 그래서 크림치즈는 냉장 상태로 바른다.
- **전자레인지 딱 10초** — 크림치즈가 살짝만 녹아 빵·잼과 어우러지는 지점. 오래 돌리면 눅눅·물러짐.
- 잼은 **저당(低糖) 제품**이 크림치즈의 산미와 균형이 좋다.
**실패 포인트:** 식빵을 안 굽고 눅눅하게 / 레인지 과다.

## 2. 길거리 계란 토스트 (韓国ストリート卵トースト)
한국 길거리·"이삭토스트"풍. **핵심은 '설탕'** — 계란전에 설탕을 뿌리는 단짠 조합이 정체성.
**핵심:**
- 계란물에 **채썬 양배추를 듬뿍** 넣어 부치면 아삭하고 촉촉한 두툼한 계란전이 된다.
- 식빵·계란전 모두 **버터**에 구워 고소함을 낸다.
- 조립: 식빵 → 계란전 → (슬라이스 치즈) → **설탕 솔솔** → 케첩·마요 지그재그 → 덮기.
**실패 포인트:** 양배추 물기 → 계란전이 질척. 채썰어 바로 부치기.

## 3. 식빵 피자 (食パンピザ)
오븐 없이 **오븐토스터/에어프라이어**로 굽는 초간단 피자.
**핵심:**
- **치즈를 토핑 아래·위로 나눠** 깔면 토핑이 빵에 붙고 고르게 녹는다.
- 식빵 **테두리를 눌러 오목하게** 만들면 소스가 흐르지 않는다.
- 남은 채소·소시지 처리에 최적. 케첩+약간의 설탕/올리고당이면 피자소스 대용.
**실패 포인트:** 소스 과다로 눅눅 → 얇게. 굽는 시간 3~5분, 치즈 녹고 노릇하면 끝.

## 4. 프렌치토스트 (フレンチトースト) — 앱 기존
**핵심:** 계란물(계란+우유+설탕) 비율과 **충분히 적시기**. 두꺼운 식빵(4~5枚切り)을 계란물에 앞뒤로 담가 **약불에서 천천히** 구워 속까지 촉촉하게. (상세는 앱 레시피)
